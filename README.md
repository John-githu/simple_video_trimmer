# 视频剪裁项目说明

## 项目目标
本项目提供一个本地可视化视频剪裁工具：
- 从“原视频目录”读取视频文件
- 在网页中预览视频并设置开始/结束时间
- 支持单个与批量剪裁
- 使用 `ffmpeg -c copy` 流拷贝，不重编码

## 当前有效代码位置
- `app/`：后端与前端核心代码
- `tests/`：自动化测试
- `运行可视化.bat`：Windows 一键启动脚本
- `requirements.txt`：Python 依赖

## 运行环境
- Windows 10/11
- Python 3.8+
- 可用的 `ffmpeg` 与 `ffprobe`（优先从 `PATH` 或项目内 `bin/` 查找）

## 目录约定（默认）
- 原视频目录：项目根目录下 `原视频/`
- 输出目录：项目根目录下 `新视频/`

目录可以在网页里重新设置，并会持久化保存。

## 部署与启动步骤（下载全部代码）
1. 安装依赖：
   - `pip install -r requirements.txt`
2. 启动服务：
   - 双击 `运行可视化.bat`
   - 或命令行执行：`python app/server.py`
3. 打开浏览器访问：`http://127.0.0.1:5000`

## 使用流程
1. 在页面顶部设置“原视频目录”（可输入路径或点击“选择目录”）。
2. 如有需要，设置“输出目录”（默认 `新视频/`）。
3. 从视频列表选择目标视频。
4. 在时间轴设置剪裁起止点。
5. 执行“单个视频剪裁”或“批量剪裁”。
6. 在“输出目录”查看结果。

## 功能说明
- 视频列表按名称升序显示
- 支持删除单个列表项
- 支持列表显示数量调整
- 支持打开原视频目录与输出目录
- 支持关闭浏览器后清理会话缓存记录


## 注意事项
- 本工具不做转码，剪裁速度快，但某些编码在非关键帧处剪裁时可能出现少量时间误差，这是流拷贝模式的常见行为。
- 若目录选择弹窗不可用，请手动输入目录绝对路径并保存。
